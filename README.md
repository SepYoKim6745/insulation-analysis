# 절연성능 평가 시스템

절연성능 평가 및 절연저항 열화 패턴 분류를 위한 웹 애플리케이션입니다.

## 기능

### 1. 절연성능 평가 모드
- **입력**: 전류(I), 온도(T)
- **계산 항목**:
  - 전기적 스트레스 (ΔI = Imax / Icritic)
  - 열적 스트레스 (ΔT = Tmax / Tcritic)
  - 온도반응 민감도 (R = ΔT / ΔI)
- **출력**: 
  - 각 항목별 위험도 평가 (4-Level)
  - 위험도에 따른 체크리스트

### 2. 절연저항 열화 패턴 분류 모드
- **입력**: 절연저항 (MΩ), 이전 측정값 (선택), 측정 횟수 (선택)
- **분류 패턴**:
  - 임계형 (Critical)
  - 가속형 (Accelerated)
  - 완만형 (Gradual)
  - 국부형 (Localized)
  - 안정형 (Stable)
- **출력**: 
  - 열화 단계
  - 관리 방향

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 상단의 모드 버튼으로 원하는 모드 선택
3. 각 모드에 맞는 입력값 입력
4. 계산 버튼 클릭하여 결과 확인

## 계산식

### 절연성능 평가
- **회귀식**: T = 39.452 + 0.025 × I + 0.014 × I²
- **Icritic**: 허용온도(70℃) 도달 전류 (회귀식 역산)
- **Tcritic**: 70℃ (허용온도)

### 위험도 기준
- **전기적 스트레스**: L1(<1.0), L2(1.0~1.2), L3(1.2~1.5), L4(≥1.5)
- **열적 스트레스**: L1(<0.5), L2(0.5~0.8), L3(0.8~1.0), L4(≥1.0)
- **온도반응 민감도**: L1(<0.4), L2(0.4~1.0), L3(1.0~1.5), L4(≥1.5)
